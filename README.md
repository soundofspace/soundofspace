# Brecht Stamper

I build Rust networking systems: proxies, gateways, protocol tooling, and performance-critical infrastructure.

Core maintainer of [rama](https://github.com/plabayo/rama), an open-source Rust framework for building proxies, servers, and clients with full control over the network stack.
